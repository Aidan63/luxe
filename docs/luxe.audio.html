Luxe Audio API

To create and play a sound, you can use the following, for ogg and wav files : 

Luxe.audio.create('clicksound', 'assets/sounds/click.wav');
Luxe.audio.volume('clicksound', 0.6);
Luxe.audio.play('clicksound');
